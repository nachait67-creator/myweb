HTML & CSS: The Web Pillars

1. HTML5 Semantic Tags

การใช้ Tag ให้ถูกความหมายเพื่อ SEO และการเข้าถึงข้อมูลที่ดีขึ้น

<main>
    <article>
        <header>
            <h2>Blog Title</h2>
        </header>
        <p>เนื้อหาบทความ...</p>
    </article>
</main>

2. CSS Flexbox & Grid

การจัดเลย์เอาต์หน้าเว็บให้สวยงามและยืดหยุ่น

/* Flexbox สำหรับจัดแถวเดี่ยว */
.container {
    display: flex;
    justify-content: space-between;
}

/* Grid สำหรับจัดตาราง 2 มิติ */
.gallery {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

3. Responsive Design (Media Queries)

การปรับแต่งให้เว็บแสดงผลได้ดีทั้งบนมือถือและคอมพิวเตอร์

@media (max-width: 600px) {
    .center-nav {
        flex-direction: column;
    }
    .content {
        padding: 10px;
    }
}
