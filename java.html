3. Stream API

จัดการข้อมูลแบบ Functional ในรูปแบบที่กะทัดรัด

List<Integer> nums = Arrays.asList(1, 2, 3, 4, 5);
int sum = nums.stream()
    .filter(n -> n % 2 == 0)
    .mapToInt(n -> n * 2)
    .sum();
