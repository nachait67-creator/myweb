C#: The Power of .NET

1. Data Types & OOP

ภาษา C# เน้นความปลอดภัยของข้อมูล (Strongly-typed) และการสืบทอด Class

public class Player {
    public string Name { get; set; } // Property
    private int score = 0;

    public void AddScore(int points) {
        score += points;
        Console.WriteLine($"{Name} has {score} points.");
    }
}

2. LINQ (Language Integrated Query)

เครื่องมือที่ช่วยให้การ Query ข้อมูลจาก List หรือฐานข้อมูลทำได้ง่ายเหมือน SQL

using System.Linq;

List<int> numbers = new List<int> { 1, 5, 8, 12, 15 };
var result = numbers.Where(n => n > 10).OrderBy(n => n);
// ผลลัพธ์: 12, 15

3. Async/Await (Multitasking)

ช่วยให้โปรแกรมไม่ค้างขณะทำงานหนัก

public async Task<string> LoadWebPageAsync(string url) {
    HttpClient client = new HttpClient();
    return await client.GetStringAsync(url);
}
